1. Pilares de um Algoritmo 
Todo algoritmo, para ser válido, deve seguir rigorosamente estas três etapas: 
1. ENTRADA: Coleta de dados (Ex: Leia, Receba). 
2. PROCESSAMENTO: Ação, cálculo ou decisão (Ex: Calcule, Verifique). 
3. SAÍDA: Resultado final para o usuário (Ex: Exiba, Escreva).
